  • Linear Regression and Gradient Descent

    This project used linear regression to find the corralation between different features (such as the number of rooms, locations, etc.) and housing prices in the Boston real estate market. The dataset was split into two: training and test datasets. The training dataset was used to build regression models, while the test dataset was used to measure the accuracy of the model. In addition, the gradient descent algorithm was was used to find the best value of beta and alpha, which helped minimize the cost function.

    Data Set: Boston Housing Prices

    Data Model: Linear/Logistic Regression

    Algorithms: Cross Validation to train the model; Gradient Descent to minimize the cost function

    Please check out the code from Github: Gradient Descent and Regression
